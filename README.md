# FriskStudio Code – UnrealEngine Edition  
Unreal Engine 専用のブラウザ IDE。

このエディションは **Unreal Engine 関連の言語だけ** に特化しています。  
Web 言語や一般的なプログラミング言語は含まれていません。

---

##  特徴

### ✔ Unreal C++ テンプレート内蔵
以下のテンプレートをワンクリックで切り替え可能：

- Actor  
- Component  
- Pawn  
- Character  
- GameMode  
- BlueprintFunctionLibrary  
- Widget（UMG）

### ✔ UE 設定ファイルの編集に対応
- `.Build.cs`  
- `.Target.cs`  
- `DefaultEngine.ini`  

### ✔ Blueprint Viewer（テキスト + ノードビュー）
- 左側：Blueprint テキスト  
- 右側：自動生成されるノードビュー  
- Event / Branch / Sequence / PrintString を解析してノード化

### ✔ Monaco Editor ベース
- シンタックスハイライト  
- 自動レイアウト  
- ダーク / ライトテーマ切替  
- ファイルダウンロード機能（DL ボタン）

---

## 🧩 使い方

1. 起動画面で編集したい UE 言語を選択  
   - Unreal C++  
   - UE Build.cs  
   - UE Target.cs  
   - UE INI  
   - Blueprint  
2. エディタ画面が開く  
3. Unreal C++ の場合はテンプレートをプルダウンで変更可能  
4. Blueprint の場合はノードビューが自動更新  
5. 「DL」ボタンでファイルを保存  

---

## 📂 対応言語

| 言語 | 説明 |
|------|------|
| Unreal C++ | Actor / Component / Pawn / Character / GameMode / BFL / Widget |
| UE Build.cs | モジュール設定 |
| UE Target.cs | ターゲット設定 |
| UE INI | エンジン設定ファイル |
| Blueprint | テキスト + ノードビュー |
